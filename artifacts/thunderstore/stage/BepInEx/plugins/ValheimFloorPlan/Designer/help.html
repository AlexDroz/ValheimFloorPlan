Valheim Floor Plan Designer v1.0.0 - User Guide

Web version (.vfp compatible)

Create and edit floor plans, then save them as .vfp files for your Valheim mod workflow.

Quick Start

  1. Set grid size with Cols and Rows, then click Apply Grid Size.
  2. Select a piece type using the radio buttons.
  3. Move mouse over the grid to preview placement.
  4. Use arrow keys while hovering the grid to rotate placement (90-degree steps).
  5. Left click to place. Right click to remove the top-most piece at that cell.

File Operations

  • New: clears current design.
  • Clear: removes all pieces from the current grid but keeps current rows and columns.
  • Open: loads a .vfp file.
  • Save: writes to the currently opened/saved file handle.
  • Save As: choose a new output file.

In Chrome/Edge, the app can remember your last folder for Open/Save dialogs.

Shell Layout Button

The Shell button builds a ready-made perimeter layout:

  • fills the area with Floor pieces,
  • adds centered doorways on each edge,
  • adds edge walls with correct rotation,
  • adds corner pillars and doorway-adjacent pillars.

Visual Rules

  • Layer order: Floor, Wall, Doorway, Pillar.
  • Overlap warnings: red outline appears on top-most overlapping pieces.
  • Placement preview: lighter transparent shade of selected piece color.

Optional Localhost Mode

You can run this app in two ways:

  • file:///... (double-click index.html) - simplest startup, basic functionality works.
  • http://localhost (local web server) - recommended for best file handling behavior.

Why localhost helps:

  • Enables secure-context browser APIs such as the File System Access API.
  • Improves Open/Save dialog behavior, including better last-folder recall.
  • Provides more desktop-like save workflows with persistent file handles.

If you do not use localhost:

  • The app still runs and the editor works.
  • File handling may fall back to simpler browser upload/download flows.
  • Folder memory and direct overwrite behavior can be limited or inconsistent.

Tip: Use StartLocalhost.bat in the web folder to launch localhost quickly.
